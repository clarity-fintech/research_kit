# x402 Protocol

Payment-gated HTTP access on CLRTY L1 — turn resource access into settled value.

## Flow

```
Request → Payment (uclrty) → Access granted → Execution
```

## Client scaffold

```javascript
const client = new x402Client();
await client.request({ resource: '/api/data', payment: 0.01 });
```

Frontend: [`frontend/products/x402.html`](../../frontend/products/x402.html)
